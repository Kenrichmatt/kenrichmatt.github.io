写在前面

18/6/17 00:21:58

在提笔的时候想了很多，做了很多前期的工作。前段时间电脑坏了，期间又掺杂了许多零零碎碎的事情，都被一些事情暂时地搁置了。一晃到了6月份，在雨天天还是占据南方大多数的日子里，世界杯已经悄然而至。经过3年多时间角逐，最终在3年后的今天这32支球队终于冲破重重关卡踏上了这有着1709万平方公里的土地。

今年注定是不平静的一年，属于全世界的足球精灵正式宣布离开；那张滑跪三道杠曾经被无数人用来做表情包的魔兽也说他就要离开了；低调的卡神也说即将结束12载红魔生涯；永远的世界第二门将Gigi在前几天也笑着和我们说：再见！斑马，曾经的欧洲三大苦也只剩伊丽莎白女王在苦苦撑着；世界地图中那片战靴模样的地图再也不会在这张图中亮起…

说好的
我们要一起征服世界
为何你在中途远走
说好的
我们要不离不弃直到彼岸
为何你独自轻舟远航
之后我们的相遇
都会是久别重逢
关于久别重逢的故事
他们之所以能够冲撞到我们内心最柔软的角落
只因有关之前相处的回忆那么美好
回忆如此玄妙
她总会抹去坏的，放大好的
我们要感谢这份玄妙
这样
我们才有怀念的理由
我们才有久别重逢的意义

━━天下足球

关于足球，关于生活，纵使有百般的不如意，百般的失望，我们还是继续坚持地爱下去！
相信，总有一天会变得更美好！

Matt
